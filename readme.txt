Professional delivery cost calculation for postcodes with WooCommerce shipping integration.

== Description ==

Postcode Delivery Calculator allows you to define delivery zones based on UK postcodes and calculate delivery costs accordingly.

Features:
* Create multiple delivery zones
* Define postcodes for each zone
* Set delivery costs and free shipping thresholds
* Exclude specific postcodes
* Collection option

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/postcode-delivery-calculator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > Shipping > Postcode Delivery to configure.
